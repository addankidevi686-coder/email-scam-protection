### Django + Machine Learning | Full Stack Web Application

---

## 🚀 Quick Start (5 Minutes)

### Step 1 — Install Python
Requires Python 3.9 or higher.  
Download from: https://python.org/downloads/

### Step 2 — Create Virtual Environment
```bash
cd email_scam_protection
python -m venv venv

# Activate on Windows:
venv\Scripts\activate

# Activate on macOS/Linux:
source venv/bin/activate
```

### Step 3 — Install Dependencies
```bash
pip install -r requirements.txt
```

### Step 4 — Train the ML Model
```bash
python train_model.py
```
This trains a Naive Bayes / Logistic Regression model on sample spam data and
saves `spam_model.pkl` and `vectorizer.pkl` into `detector/ml_model/`.

> **Pro Tip:** For higher accuracy, download the UCI SMS Spam Collection dataset
> from https://archive.ics.uci.edu/ml/datasets/SMS+Spam+Collection and run:
> ```bash
> python train_model.py path/to/SMSSpamCollection
> ```

### Step 5 — Apply Database Migrations
```bash
python manage.py migrate
```

### Step 6 — Create Admin Superuser
```bash
python manage.py createsuperuser
```
Follow the prompts to set username, email, and password.

### Step 7 — Run the Development Server
```bash
python manage.py runserver
```
Open your browser at: **http://127.0.0.1:8000/**

## 🔌 API Usage

### POST `/api/check/`
Detect spam without authentication.

**Request:**
```bash
curl -X POST http://127.0.0.1:8000/api/check/ \
  -H "Content-Type: application/json" \
  -d '{"email_text": "Congratulations! You won a FREE prize! Claim NOW!"}'
```

**Response:**
```json
{
  "result": "spam",
  "confidence": 87.5,
  "is_spam": true
}
```

## 🎨 Features Overview

- ✅ **User Authentication** — Register, Login, Logout with hashed passwords
- ✅ **Email Spam Detection** — ML-powered with Naive Bayes / Logistic Regression
- ✅ **Confidence Score** — See how sure the model is (0–100%)
- ✅ **Keyword Highlighting** — Suspicious words highlighted in yellow
- ✅ **File Upload** — Upload .txt email files
- ✅ **AJAX Quick Check** — Real-time detection without page reload
- ✅ **History Page** — View, filter, and delete past checks
- ✅ **Admin Panel** — Manage users and email records
- ✅ **Dark Mode** — Toggle with one click, persists via localStorage
- ✅ **REST API** — Public endpoint for programmatic access
- ✅ **Responsive Design** — Works on mobile, tablet, desktop
- ✅ **Sample Emails** — Built-in spam/safe examples to test with

## 🔧 Troubleshooting

**ModuleNotFoundError: No module named 'django'**
→ Make sure your virtual environment is activated and you ran `pip install -r requirements.txt`

**Model not found warning**
→ Run `python train_model.py` first. Without it, the system uses rule-based detection.

**Port already in use**
→ Run on a different port: `python manage.py runserver 8080`

**Static files not loading**
→ Run: `python manage.py collectstatic`
